## What is FOV?

**Field of View (FOV)** is how wide your view is in a game, measured in degrees. A higher FOV shows more of the game world but can distort visuals at the edges. A lower FOV is more zoomed in, which some players prefer for precision.

### Horizontal vs Vertical FOV

Games use different FOV measurement axes. Most PC games use **horizontal FOV**.

```
Vertical FOV = 2 × arctan(tan(Horizontal FOV / 2) / Aspect Ratio)

Example (16:9, 90° horizontal):
VFOV = 2 × arctan(tan(45°) / 1.778)
VFOV ≈ 59°
```

## FOV & Sensitivity Relationship

Changing your FOV affects how fast your crosshair moves across targets. A wider FOV means your sensitivity feels lower (crosshair covers more screen per mouse movement but the same physical targets appear smaller).

### Scaling Sensitivity with FOV Changes

```
New Sensitivity = Old Sensitivity × (tan(New FOV / 2) / tan(Old FOV / 2))

Example: Going from 90° → 103° FOV
New Sens = Old Sens × (tan(51.5°) / tan(45°))
New Sens = Old Sens × 1.233
```
